# javajam_coffee_bar
Website for a fictional coffee shop, with focus on layout navigation and page structure.

---

### Live site  
https://diegomartins8.github.io/javajam_coffee_bar/ 

---

### Tools used
- HTML  
- CSS  

---

### Description

This site was built for a Web Design course assignment and is based around a small fictional coffee shop that hosts live music and is always looking for new staff.

The site contains 4 pages:

- Home page introduces the coffee shop, their features, and address/number 
- Menu page lists the beverage menu
- Music page displays the scheduled live performances and includes audio playback for each artist
- Jobs page lists open roles and basic job details  

---

### Skills

- Page structure built with HTML  
- All styling through a single CSS file, with consistent colors for the header footer and backgrounds  
- Images stored in a folder and sized to fit layout  
- Pages connected through text links along with an audio element on the music page
